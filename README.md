# Clibo Site

Official website for Clibo.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project Structure

- `index.html` - landing page
- `src/styles.css` - site styles
- `public/assets/` - static assets served from `/assets`
- `docs/direct-sales-licensing.md` - Dodo Payments direct-sales licensing plan

## Payment Integration Notes

The current Pro purchase link is a placeholder. Replace it with a Dodo Payments checkout URL or a backend-created checkout session URL when the product is configured.
